# 형상모델링 퀴즈 — 선생님 할 일 (최소)

**학생 링크:** https://shape-modeling-quiz.vercel.app

---

## Supabase까지 “한 번에” (추천)

1. `shape-modeling-quiz` 폴더에 **`.env.local`** 만들기 (내용은 **`.env.local.example`** 참고)
2. Supabase에서 **URI·Project URL·service_role** 3개만 복사해 붙이기
3. 그 폴더에서 터미널:

```bash
npm install
npm run setup:supabase
```

→ 테이블 생성 + Vercel 변수 + 재배포까지 자동.

---

## 수동 (웹만 쓸 때)

| 1 | Supabase SQL Editor에서 `supabase/quiz_scores.sql` 실행 |
| 2 | Vercel에 `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY` → Redeploy |
