## 처음 딱 한 번

| # | 하는 일 | 시간 |
|---|---------|------|
| **1** | Supabase 로그인 → 프로젝트 → **SQL Editor** → 저장소 `supabase/quiz_scores.sql` 전체 붙여넣고 **Run** | 1분 |
| **2** | 같은 Supabase → **Project Settings → API** → `URL` + `service_role` 복사 | 30초 |
| **3** | Vercel → **shape-modeling-quiz** → **Settings → Environment Variables**에 붙여넣기 → **Save** → **Deployments → Redeploy** | 1분 |

**Vercel 변수 이름 (딱 두 개)**

- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
