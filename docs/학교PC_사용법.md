# 학교 PC 전용 모드

인터넷(Vercel) 없이 **교실 PC에서만** 시험을 보게 할 때 사용합니다.

## 준비 (최초 1회)

1. 이 PC에 [Node.js](https://nodejs.org) 설치
2. 프로젝트 폴더에서 `npm install` (시작 배치가 자동 실행)

## 수업 때

1. 바탕화면 **`형상모델링검토_학교PC_시작.bat`** 더블클릭
2. 검은 창에 나온 주소를 학생에게 알려줌  
   예: `http://192.168.0.15:3000`
3. 학생은 **로그인 없이** 학번·이름 입력 후 제출
4. 제출할 때마다 바탕화면 **`형상모델링검토_성적.csv`** 에 한 줄 추가  
   → 엑셀로 바로 열 수 있음

## 주의

- 선생님 PC와 학생이 **같은 와이파이/학교 내부망**이어야 함
- 서버 창을 끄면 접속 불가 (수업 내내 켜 두기)
- 방화벽에서 3000 포트 허용이 필요할 수 있음

## Vercel(인터넷)과 차이

| 항목 | 학교 PC 모드 | Vercel |
|------|-------------|--------|
| 접속 | `http://교실PC IP:3000` | `shape-modeling-quiz.vercel.app` |
| 성적 | 바탕화면 CSV | Google 스프레드시트 |
| 스프레드시트 연동 | **불필요** | 웹앱 URL 필요 |
